Petfinder

Visit petfinder-app.herokuapp.com

  • Web development
  • Web design

This project is a simple MEAN stack implementation of the Petfinder website created for my web development course. Connecting to the Petfinder API, the app allows users to search for local adoptable pets, filter the results, save their favorites, create profiles and add friends. This was my first experience building a dynamic web app with a server, API and user authentication. Check out the source code on GitHub or watch my code walkthrough video.
